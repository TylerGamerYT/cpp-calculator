# 🧮 C++ Scientific Calculator

Simple terminal calculator written in C++.

## 🚀 Features
- Basic math operations
- Power + square root
- sin, cos, tan (RAD/DEG mode)

## ⚙️ Run
g++ main.cpp -o calc  
./calc
